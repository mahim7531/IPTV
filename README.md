# 📺 IPTV - Live TV Streaming Website

A modern and responsive **IPTV Live TV Player Website** built with HTML, CSS, and JavaScript.  
This project provides a smooth live streaming experience with multiple channel categories, HLS video playback, and a modern dark UI design.

---

## 🚀 Features

- 📺 Live TV Streaming Player
- 🎬 HLS (.m3u8) Stream Support
- 🏆 Sports Channels
- 🇧🇩 Bangla Channels
- 🇮🇳 Hindi Channels
- 📰 News Channels
- 🎵 Music Channels
- 🧒 Kids Channels
- 🌎 English Channels

### UI Features

- ✨ Modern Glassmorphism Design
- 🌙 Dark Theme Interface
- 🎨 Neon Gradient Effects
- 📱 Fully Responsive Design
- 💻 Desktop & Mobile Friendly
- 🔥 Smooth Channel Switching
- 🎯 Bottom Navigation Menu
- 🖼️ Channel Logo Display


---

# 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- HLS.js
- Netlify Hosting
